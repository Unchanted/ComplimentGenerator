# Compliment generator

This project is a React-based application that simulates a spinning wheel with a typewriter effect. The spinning wheel, when clicked, rotates and then displays a random compliment with a typewriter-style animation. This project also includes animated background beams for added visual flair.

## Features

- **Spinning Wheel**: A colorful, animated wheel that spins when the user clicks the "Spin the Wheel" button.
- **Typewriter Effect**: After the wheel stops spinning, a random compliment is displayed using a typewriter animation.
- **Background Beams**: Dynamic, animated background beams that enhance the visual appeal of the page.
- **Responsive Design**: The application is designed to work seamlessly on different screen sizes.


## Installation

   ```bash
   git clone https://github.com/Unchanted/ComplimentGenerator.git
   cd ComplimentGenerator
   npm install
   npm run dev
   ```
